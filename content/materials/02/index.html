In this session, you will:

  1. Knit the same document to different built-in output formats (Word and PDF) using the knit button.

  2. Knit the same document to different built-in output formats (Word and PDF) using the YAML to store output options.

  3. Knit the same document to a different output format from an extension package (distill::distill_article) by changing the YAML.

  4. Make a markdown table.

  5. Make a table one.

Output formats used

  • HTML
  • Word
  • PDF
  • distill::distill_article
