Episodes

  1. Metadata

  2. Text

  3. Code

  4. Output

In this session, you will:

  1. Identify the basic anatomy of an R Markdown document.

  2. Use the YAML to set up meaningful metadata and change output format options.

  3. Organize and style your text using markdown formatting.

  4. Organize your code using knitr chunk labels. (–>put in workflows graphics?)

  5. Style your code’s output using knitr chunk options.

  6. Knit a document with updated data.
